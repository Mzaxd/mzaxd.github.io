为什么设置代理？

在使用Git操作远程库（Github）的时候偶尔会失败，通常都是网络的问题（众所周知的原因），通过使用代理可以有效避免这个问题

查看代理本地监听的端口号

  1. 打开v2rayN的管理面板
  2. 点击设置，选择参数设置
  3. 在弹出的参数设置页面找到下图中的本地监听端口和协议
  4. 根据你的端口号和协议进行对应设置
